# Tony | Video Editor & Multimedia Specialist Portfolio

A premium, cinematic portfolio showcasing high-end video editing, motion design, and visual storytelling.

![Portfolio Preview](Assets/Picture/Tony%20Portrait.PNG)

## ✨ Features

- **Cinematic Loading Screen**: A sophisticated staggered intro with theme-switching and real-time counter.
- **Dynamic Hero Section**: Interactive GSAP-driven video expansion and text-merging scroll animations.
- **Asymmetric Portfolio Grid**: A curated archive of selected works with hover-to-play video previews.
- **Premium Aesthetics**: Editorial-style typography (Anton & Space Grotesk) with a minimalist cream-and-dark color palette.
- **Fully Responsive**: Optimized for all devices using Tailwind CSS.
- **Interactive Modals**: Seamless video playback for featured projects.

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/) with ScrollTrigger
- **Typography**: [Google Fonts](https://fonts.google.com/) (Anton, Space Grotesk)

## 🚀 Getting Started

### Prerequisites

You need a local web server to run this project correctly due to CORS requirements for local video assets.

### Installation & Running

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. Navigate to the project directory:
   ```bash
   cd your-repo-name
   ```
3. Start a local server:
   Using Python:
   ```bash
   python3 -m http.server
   ```
   Or using Node.js `serve`:
   ```bash
   npx serve
   ```
4. Open your browser and navigate to `http://localhost:3000` (or the port provided by your server).

## 📂 Project Structure

- `index.html`: Core structure and SEO metadata.
- `style.css`: Custom design tokens and micro-interactions.
- `script.js`: GSAP animations and interactive logic.
- `Assets/`: High-quality video and image assets.
